О программе

NetControl
NETWORK CONTROL PANEL  ·  v1.0.0  ·  by CHELOVECHEK
Учебный инструмент для анализа, мониторинга и диагностики сети.
Создан исключительно в познавательных целях.

Возможности

▸ Сканирование портов (одиночное и массовое)
▸ Ping и Traceroute
▸ DNS Lookup (A, MX, NS, TXT, CNAME)
▸ WHOIS информация о домене
▸ Геолокация IP адреса
▸ Проверка SSL сертификатов
▸ Мониторинг сетевых соединений
▸ Калькулятор подсетей (CIDR)
▸ Сетевые утилиты
▸ Стресс-тестирование (HTTP / TCP / ICMP)

Стек технологий

Python 3 Flask · Flask-SocketIO · python-whois · dnspython
JavaScript Vanilla JS · Socket.IO client
CSS Custom · FiraCode · FiraMono · UnitedSans
HTML Jinja2 templates
Audio eDEX-UI sound pack (MIT)
Transport WebSocket · REST API
